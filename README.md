# Daily-Journal

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Daily-Journal is a blog app built with Node.js, Express, Lodash, and EJS that allows users to create a journal of their daily activities. It provides a simple and intuitive interface for users to record their thoughts, experiences, and reflections.

## Features

Create Journal Entries: Users can create new journal entries by adding a title, date, and content. They can document their daily activities, thoughts, goals, achievements, or any other information they want to record.

View Entries: Users can view their journal entries in a clean and organized manner. Entries are displayed in reverse chronological order, making it easy to navigate through previous days' entries.

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install the dependencies: npm install

## Usage

Start the app: npm start
Open your web browser and navigate to http://localhost:3000.
Start creating and managing your journal entries.
